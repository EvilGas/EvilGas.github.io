Markdown基本语法

Created2025-08-18|Updated2025-08-24
|Word Count:1.1k|Reading Time:4mins|Post Views:

Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写，Markdown被大量使用，如Github、Wikipedia、简书等。Mardown语法速查

Markdown 标题语法

要创建标题，请在单词或短语前面添加井号#，#的数量代表了标题的级别

Markdown 段落语法

要创建段落，请使用空白行将一行或多行文本进行分隔。不要用空格（spaces）或制表符（ tabs）缩进段落

Markdown 换行语法

在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行。幸运的是，几乎每个Markdown应用程序都支持另一种换行方式：HTML的<br>标签

Markdown 强调语法

斜体

要用斜体显示文本，请在单词或短语前后添加一个星号*

粗体

要加粗文本，请在单词或短语的前后各添加两个星号*

斜体和粗体

要同时用粗体和斜体突出显示文本，请在单词或短语的前后各添加三个星号*

删除线

若要删除单词，请在单词前后使用两个波浪号~~

Markdown 引用语法

要创建块引用，请在段落前添加一个>符号

Markdown 列表语法

有序列表

要创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点

无序列表

要创建无序列表，请在每个列表项前面添加破折号-并缩进一个

Markdown代码语法

要将单词或短语表示为代码，请将其包裹在反引号` 中

如果你要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号`` 中

如果想要书写一段代码，使用一组``` 来进行代码块标记

Markdown 分隔线语法

要创建分隔线，请在单独一行上使用三个或多个破折号---，并且不能包含其他内容。

Markdown 链接语法

超链接Markdown语法代码：文本[超链接显示名](超链接地址 "超链接title")，链接title可选

使用尖括号<>可以很方便地把URL或者email地址变成可点击的链接

Markdown 图片语法

插入图片Markdown语法代码：![图片alt](图片链接 "图片title")

给图片增加链接，请将图像的Markdown括在方括号中，然后将链接添加在圆括号中

Markdown 转义字符语法

要显示原本用于格式化 Markdown 文档的字符，请在字符前面添加反斜杠字符\

Markdown 表格语法

要添加表格，请使用三个或多个连字符---创建每列的标题，并使用管道|分隔每列，同时设置对齐格式，:---为左对齐，:---:为居中对齐，---:为右对齐
